# Описание Агента для Поиска Контактов (Версия 1.1)

Изменения vs v1.0:
- Источники: добавлены Google Maps/Places (сайты), Chabad Locator, JCC Association, каталоги WJC/JDC, муниципальные каталоги
- Соцсети: LinkedIn Organizations, Facebook Pages About, Instagram bio → сайт
- Качество: нормализация доменов, дедуп по (org+domain|host), приоритет целевых email (education@, youth@)
- Верификация: формат + MX (без отправки), мягкий SMTP‑пинг (ограниченно)
- Обогащение: Language, Type классификация по ключевым словам, активность по соцсетям
- Логирование: поля `SourceDetail`, `Confidence`, `DiscoveredAt`, `VerifiedAt`
- Стоимость: батчи, кэш доменов, параллельный I/O, лимиты внешних проверок

Процесс (кратко):
1) Систематический поиск по приоритетам источников
2) Извлечение сайта → извлечение email/контактов
3) Валидация (формат, MX), классификация, язык
4) Дедуп и объединение с существующей базой
5) Документирование: поля источника, даты, confidence
6) Контроль качества: выборочная проверка, отчёт

Результат: новые и обогащённые записи, готовые к outreach.
