## Usage Guidelines

### Template Selection
- **Template 1**: Quick recommendation to acquaintances
- **Template 2**: Personal introduction from developer
- **Template 3**: Direct feedback request for engaged users
  - **Variant A** ‚úÖ: Currently used - includes Advisors group invite (best for active users)
  - **Variant B**: Short & direct (best for busy people)
  - **Variant C**: Founder introduction (best for first-time contact)
  - **Variant D**: Specific questions (best for detailed feedback)
  - **Variant E**: Value-focused (best for engaged users)
  - **Variant F**: Casual (best for younger audience)
- **Template 4**: Torah chat groups - Non-intrusive, curiosity-driven, highlights unique value

### Best Practices
- Always personalize with user's name when possible
- Choose variant based on:
  - User's engagement level
  - Communication style (formal vs casual)
  - Desired response depth
  - Whether you want to invite to Advisors group
- Track which template/variant was used in `summary.md`
- A/B test different variants to see what works best
