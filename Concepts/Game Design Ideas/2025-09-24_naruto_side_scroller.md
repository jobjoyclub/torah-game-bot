# Шаблон идеи
- Название: 2D Pixel Side-Scroller — Ninja Jutsu Run
- Проблема / возможность: Добавить динамичный экшен-формат, повышающий удержание и реиграбельность.
- Идея (1–2 предложения): Спроектировать 2D пиксельный сайд-скроллер во вселенной ниндзя (в стиле популярного аниме) с управлением персонажами наподобие Kakaxi, каждый со своими уникальными техниками (jutsu), бегом по стенам и ультимативными способностями. Уровни — узнаваемые локации вроде Hidden Leaf Village, с противниками-ниндзя.
- Как работает в игре (флоу): Игрок бежит слева направо, использует базовые атаки и комбинации jutsu, преодолевает препятствия (стены, крыши — wall-running), побеждает вражеских ниндзя, собирает бустеры чакры для активации ульты; боссы — на концах ключевых локаций.
- Учебная ценность / связь с митцвот: Можно адаптировать моральные дилеммы и ценности (дисциплина, товарищество, ответственность) через миссии и диалоговые сцены между уровнями.
- Риски / ограничения: Использование реальных имен/символики может нарушать IP; лучше применить оригинальный сеттинг и отсылки без прямого использования лицензированных персонажей и названий.
- Метрика успеха: Средняя длительность сессии, прогресс по уровням, процент завершения боссов, частота использования ульты, удержание D1/D7.
- Заметки для реализации: 
  - Визуал: 8-bit/16-bit пиксель-арт, плавные партиклы для jutsu.
  - Механики: dash, двойной прыжок, wall-run, парирование, комбо.
  - Контент: 3–5 биомов (деревня, лес, крыши, горные тропы, храм), мини-боссы и финальные боссы.
  - Технически: Canvas/WebGL, 60 FPS, поддержка мобильных экранов.
