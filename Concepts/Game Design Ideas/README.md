# Идеи геймдизайна

Здесь собираем гипотезы и задумки для улучшения сценария и игрового опыта Shabbat Runner.

Структура предложений:
- Проблема / возможность
- Идея (1–2 предложения)
- Как это работает в игре (короткий флоу)
- Ценность для игрока / обучающий эффект
- Риски и ограничения
- Метрика успеха

Новые идеи добавляйте отдельными файлами в этом же каталоге по шаблону `YYYY-MM-DD-коротко-о-чем.md`.
